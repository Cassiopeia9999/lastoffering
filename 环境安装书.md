# 失物招领系统（FastAPI）—— 环境安装说明

---

## 环境总览

| 环境名 | 用途 | 核心依赖 |
|--------|------|----------|
| `lostfound` | 后端服务运行 | FastAPI, SQLAlchemy, 无需 PyTorch |
| `opim` | 数据集下载 | fiftyone, bing-image-downloader |
| `efftrain` | 模型训练 | PyTorch + CUDA |

---

## 一、lostfound 环境（后端服务）

### 创建环境

```bash
conda create -n lostfound python=3.10.13
conda activate lostfound
```

### 安装依赖

```bash
conda activate lostfound
cd F:\Last_offering\items
pip install -r requirements-minimal.txt
pip check
```

### 分步安装（可选）

```bash
conda activate lostfound

# 基础科学计算库
pip install numpy==1.24.4
pip install pandas==2.1.4
pip install scikit-learn==1.3.2

# FastAPI 核心及工具
pip install fastapi==0.124.4
pip install uvicorn[standard]==0.33.0
pip install pydantic==2.10.6
pip install pydantic-settings==2.1.0
pip install python-dotenv==1.0.1
pip install PyYAML==6.0.1
pip install python-multipart==0.0.20

# 数据库相关
pip install sqlalchemy==2.0.25
pip install pymysql==1.1.0
pip install alembic==1.13.1

# 图像处理库
pip install pillow==10.2.0

# 通用工具库
pip install requests==2.31.0
pip install tqdm==4.66.2

# JWT
pip install python-jose[cryptography]

# 密码加密（bcrypt 必须为 4.0.1）
pip install passlib[bcrypt]==1.7.4
pip install bcrypt==4.0.1

pip check
```

> **注意**：后端服务默认使用占位模式，无需安装 PyTorch。如需启用真实 AI 模型，参考第五节。

## 二、opim 环境（数据集下载）

用于从 OpenImages 下载数据集和爬取图片。

```bash
# 创建环境
conda create -n opim python=3.9.19
conda activate opim

# 基础库
pip install numpy==1.24.4
pip install pandas==2.1.4
pip install pillow==10.2.0
pip install tqdm==4.66.2

# 数据集下载工具
pip install fiftyone==0.23.4

# 数据库依赖（fiftyone 需要）
pip install dnspython==2.7.0 motor==3.3.2 pymongo==4.5.0

# 图片爬虫工具
pip install bing-image-downloader

pip check
```

**运行数据集下载：**
```bash
conda activate opim
python ai_module/download_openimages.py
python ai_module/download_kaggle.py
python ai_module/check_dataset.py
```

## 三、efftrain 环境（模型训练）

用于训练 EfficientNet-B0 分类模型。

```bash
# 创建环境
conda create -n efftrain python=3.10.13
conda activate efftrain

# 基础库
pip install numpy==1.24.4
pip install pillow==10.2.0
pip install tqdm==4.66.2

# PyTorch GPU 版（CUDA 11.8，推荐）
pip install torch==2.2.1 torchvision==0.17.1 --index-url https://download.pytorch.org/whl/cu118

# PyTorch CPU 版（无显卡时使用）
# pip install torch==2.2.1 torchvision==0.17.1

pip check
```

**运行训练：**
```bash
conda activate efftrain
python ai_module/train.py
```

**运行测试：**
```bash
conda activate efftrain
python ai_module/predict.py <图片路径>
python ai_module/test.py
```

## 四、数据库配置

1. 启动 MySQL 服务
2. 创建数据库：
   ```sql
   CREATE DATABASE IF NOT EXISTS lostfound CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```
3. 将 `.env.example` 复制为 `.env`，填写 `DB_PASSWORD` 等配置

## 六、启用真实 AI 模型

默认情况下，后端使用占位模式（无需 PyTorch）。训练完成后，可启用真实模型：

### 方式 A：在 lostfound 环境中安装 PyTorch

```bash
conda activate lostfound
pip install torch==2.2.1 torchvision==0.17.1 --index-url https://download.pytorch.org/whl/cu118
```

### 方式 B：直接使用 efftrain 环境运行后端

```bash
conda activate efftrain
python -m uvicorn backend.app.main:app --reload --host 0.0.0.0 --port 8000
```

### 修改配置

编辑 `backend/app/services/ai_classifier.py` 和 `backend/app/services/ai_feature.py`：

```python
USING_PLACEHOLDER = False  # 改为 False
```

---

## 七、注意事项

1. **bcrypt**：必须使用 `bcrypt==4.0.1`，5.x 与 passlib 不兼容。
2. **MySQL**：启动前确认 MySQL 服务已运行。
3. **CUDA**：确保显卡驱动支持 CUDA 11.8，否则使用 CPU 版 PyTorch。
4. **环境隔离**：三个环境职责分离，避免依赖冲突。

├── requirements-minimal.txt
