## 项目成果总结

### 技术栈
- **后端**: FastAPI + SQLAlchemy + MySQL + JWT
- **前端**: Vue3 + Vite + Element Plus + Pinia
- **AI**: PyTorch + EfficientNet-B0

### 核心功能
1. **用户系统**: 注册/登录/JWT认证/权限控制/个人资料管理/头像上传
2. **物品管理**: 发布/编辑/下架/查询/分类筛选/图片裁剪/特征向量存储/重新上架
3. **AI识别**: 图片自动分类（15类）、特征提取、Top-K预测、重新识别功能
4. **以图搜物**: 基于余弦相似度的图像检索、相似度阈值调节
5. **智能匹配**: 后台自动匹配+通知推送、疑似遗失申请、认领确认
6. **留言系统**: 物品留言、留言回复、留言删除、实时通知
7. **管理后台**: 数据统计、物品/用户管理、内容审核

### AI模型
- **分类模型**: EfficientNet-B0，15类物品分类
- **特征维度**: 512维，支持以图搜图
- **数据集**: OpenImages + Bing爬虫，约7500张图片（训练集500张×15类）

### Conda环境
- **lostfound**: 系统整体运行环境（FastAPI + 前端开发服务器）
- **opim**: 数据集采集、挖掘、裁剪、分配
- **efftrain**: 模型训练、调试、功能测试环境
